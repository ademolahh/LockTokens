# Lock Tokens

     This smart contract is used to lock tokens for specific time,
     It doesnt come with reward,
     It supports Any ERC20,ERC721,ERC1155 token
     
# Build and Test
Use node version 14 or 16
```
npm install
```

# Deployemnent
```
npx hardhat deploy --network <network>
```

# Test
```
npx hardhat test
```
